====== Добро пожаловать в Академию перевода ======

The “translationAcademy” is intended to enable anyone, anywhere to learn the basics of how to translate the Bible into their own language, or help someone who wants to do so. It is intended to be highly flexible, can be in-advance or just-in-time (or both, as needed), modular, and highly visual, in web and offline mobile video formats.

===== Обзор translationAcademy =====

translationAcademy has been created with an introduction and four manuals to help answer questions pertaining to translation, checking, technical, and process.

  * [[:ru:ta:vol1:intro:toc_intro|Введение]] - Introduces unfoldingWord and translationAcademy.
  * [[:ru:ta:vol1:translate:toc_transvol1|Руководство по переводуl]] - Teaches translation theory and answers many practical translation questions.
  * [[:ru:ta:vol1:checking:toc_checkvol1_2|Руководство по редактированию, проверке]] - Teaches checking theory and answers many practical checking questions.
  * [[:ru:ta:vol1:tech:toc_techvol1_2|Руководство по технологии]] - Contains tutorials on how to use the various technology tools that we have created.
  * [[:ru:ta:vol1:process:toc_processvol1_2|Руководство по работе с материалами]] - Provides a step-by-step guide through the setup, translation, checking, publishing, and distribution phases.
  * [[:ru:ta:vol2:audio:toc_audiovol2|Руководство по работе с аудио]] - How to make high-quality audio recordings without premium gear and a recording studio
  * [[:ru:ta:vol2:gateway:toc_gatewayvol2|Руководство для языков]] - Deals with issues that come up for Gateway Languages that don't occur for other languages

==== Инструмент для визуального проектирования translationAcademy ====

Нам нужны “Общественные Эксперты”, чтобы приступить к идентификации модулей, которые необходимы в каждой области фокусировки, а затем описывающие содержание и (в ближайшее время) создания записи обучающих видео.

Готовы начать?

  * Перейти к странице [[:ru:ta:workbench|Инструмент для визуального проектирования Translation Academy]], чтобы прочитать о состоянии ваших материалов.
  * Перейти к странице [[:ru:ta:workbench:how-to-use|Как добавить мою информацию]], чтобы узнать, как добавить вашу информацию.

Если вы новичок в translationAcademy, вы также можете взглянуть на руководство по переводу, чтобы изучить основы перевода, проверки, и за его пределами!
