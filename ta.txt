====== Добро пожаловать в “Академию перевода” ======

“Академия перевода” предназначена для того, чтобы любой человек не зависимо от мета, в котором находится мог узнать основы того, как перевести Библию на его родном языке, или помочь кому-то, кто хочет сделать это. “Академия” предназначена быть 'под рукой', быть доступной в Интернете и в автономном режиме, мобильном видео формате.
===== Обзор “Академии перевода” =====

“Академия перевода” has been created with an introduction and four manuals to help answer questions pertaining to translation, checking, technical, and process.

  * [[:ru:ta:vol1:intro:toc_intro|Содержание]] - Introduces unfoldingWord and “Академия перевода”.
  * [[:ru:ta:vol1:translate:toc_transvol1|Руководство по переводу]] - Teaches translation theory and answers many practical translation questions.
  * [[:ru:ta:vol1:checking:toc_checkvol1_2|Руководство по редактированию, проверке]] - Teaches checking theory and answers many practical checking questions.
  * [[:ru:ta:vol1:tech:toc_techvol1_2|Руководство по технологии]] - Contains tutorials on how to use the various technology tools that we have created.
  * [[:ru:ta:vol1:process:toc_processvol1_2|Руководство по работе с материалами]] - Provides a step-by-step guide through the setup, translation, checking, publishing, and distribution phases.
  * [[:ru:ta:vol2:audio:toc_audiovol2|Руководство по работе с аудио]] - How to make high-quality audio recordings without premium gear and a recording studio
  * [[:ru:ta:vol2:gateway:toc_gatewayvol2|Руководство по ключевым языкам]] - Deals with issues that come up for Gateway Languages that don't occur for other languages
