===== Иезекииль 48 Вопросы по переводу =====

В? Какому колену будет дана земля на северной границе Израиля?

О. Колену Дана будет дана земля на северной границе Израиля. [48:1]

В? Какое колено будет находится южнее границы колена Дана?

О. Колено Асира будет находится южнее границы колена Дана. [48:2]

В? Какое колено будет находится южнее границы колена Асира?

О. Колено Неффалима будет находится южнее границы колена Асира. [48:3]

В? По порядку, какие четыре колена будут находится южнее границы колена Неффалима?

О. По порядку, четыре колена, которые будут находится южнее границы колена Неффалима, это Манассия, Ефрем, Рувим и Иуда. [48:4-7]

В? Что будет находиться в середине земли, расположенной вдоль границы с Иудой?

О. В середине земли, расположенной вдоль границы с Иудой, будет находится храм. [48:8]

В? Кто будет жить на земле вокруг святилища Яхве?

О. Священники по линии Садока будут жить на земле вокруг святилища Яхве. [48:11]

В? Кто будет жить на земле, граничащей с землей священников?

О. Левиты будут жить на земле, граничащей с землей священников. [48:13]

В? Для чего будет использоваться остальная территория священного участка земли?

О. Остальная территория священного участка земли будет использоваться для выращивания еды для работников этого города. [48:18]

В? Кому будет принадлежать земля к востоку и западу от границ священного участка земли?

О. Земля к востоку и западу от границ священного участка земли будет принадлежать правителю. [48:21]

В? По порядку, какие пять колен будут находится южнее священного участка земли?

О. По порядку, пять колен, которые будут находится южнее священного участка земли, это Вениамин, Симеон, Иссахар, Завулон и Гад. [48:23-27]

В? Сколько ворот будет на каждой стороне города, и что будут представлять каждые ворота?

О. Там будет по трое ворот на каждой стороне города, каждое из них будет представлять одно из колен Израильских. [48:30-34]

В? Как будет называться этот город?

О. Этот город будет называться «Яхве Здесь». [48:35]
