==== Руководство по редактированию перевода ====

Данное руководство описывает процесс проверки библейского текста на его правильность, понятность и благозвучность.

Том первый

В первом томе содержатся инструкции по редактированию перевода, которые команда переводчиков должна использовать для проверки своей работы. Выполнение данных инструкций может помочь им пройти первый уровень проверки. В первом томе также содержатся указания для команды переводчиков, которыми они могут пользоваться при тестировании текста на языковом сообществе в отношении понятности и благозвучности перевода, а также указания для руководителей церкви по тестированию перевода на его правильность. Следование данным инструкциям поможет пройти второй уровень редакции. В настоящем томе также содержатся указания для лидеров церковной рабочей сети, служащие для проверки правильности перевода в ходе третьего уровня редактирования.

Том второй

Во втором томе содержатся дальнейшие инструкции для проверки перевода, которые могут использоваться лидерами церковной рабочей сети для редактирования текста. Поскольку многие из лидеров церковной рабочей сети не владеют иностранным языком, во второй том включены инструкции по созданию обратного перевода, позволяющего сотрудникам проверять перевод с языка, которым они не владеют.
