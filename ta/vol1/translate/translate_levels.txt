==== Уровни значения ====

Для создания качественного перевода требуется, чтобы значение в целевом языке соответствовало значению в исходном языке.

В Библии существует много разных уровней значения. Вот эти уровни:

  * Значение слов
  * Значение фраз
  * Значение предложений
  * Значение абзацев
  * Значение глав
  * Значение книг

==== В словах содержится значение ====

То есть наименьшая лексическая единица - слово - обладает значением. Но это значение определяется контекстом, в котором находится данное слово. Например, слово «давать», в зависимости от контекста, может иметь следующие оттенки значения: * дарить подарок * дать трещину * сдаваться * сдать должность * предавать * снабжать и т.д.

==== Построение более широкого значения ====

Переводчик должен определить значение каждого слова в контексте, а затем воспроизвести это значение в переводимом тексте. Затем из этих слов составляются фразы, предложения, абзацы и т.д. Поэтому переводчик должен удостовериться, что слова формируют друг с другом соответствующие фразы, предложения, абзацы и т.д., и что значение текста сохраняется на всех уровнях.
