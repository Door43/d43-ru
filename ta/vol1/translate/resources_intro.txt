==== Ознакомление с материалами ====

Проект «Раскрывающееся Слово» разработал несколько материалов, которые будут вам полезны при переводе Библии или «Историй из открытой Библии». Все они доступны для всех проектов на сайте Door43, также большая их часть доступна в translationStudio (Студии перевода).

Это следующие ресурсы:

  * Истории открытой Библии
  * Открытая точная Библия (ОТБ)
  * Открытая живая Библия (ОЖБ)
  * Слова для перевода
  * Замечания по переводу
  * Вопросы перевода
  * Академия перевода

=== Что нужно знать ===

  * «Истории открытой Библии» - это 50 иллюстрированных рассказов, передающие в краткой форме библейское послание.
  * В «Историях открытой Библии» содержится своя группа слов для перевода, замечаний по переводу и вопросов для перевода.
  * ОТБ облекает язык Библии в современный, чтобы показать структуру и стиль оригинала.
  * ОЖБ представляет собой перевод, помогающий сделать библейское значение простым и легким для понимания.
  * Библейские «Слова для перевода» - это важные понятия, взятые из ОТБ.
  * Библейские «Замечания по переводу» - это объяснение текста ОТБ.
  * «Слова для перевода» - это важные термины, которые следует знать, чтобы правильно переводить.
  * «Замечания по переводу» дают дополнительную информацию о том, как сделать перевод на родной язык понятным и благозвучным.
  * «Вопросы перевода» - это ряд вопросов с ответами, которые вы можете использовать, чтобы сверить ваш перевод с речью носителей родного языка.
  * «Академия перевода» содержит информацию о том, как переводить и проверять текст, а также как использовать переводческие средства «Раскрывающегося Слова».
