==== Использование вспомогательных средств при переводе ====

Для того, чтобы помочь переводчикам осуществлять максимально качественный перевод, были разработаны «Замечания по переводу» и «Слова для перевода». Есть ряд «Замечаний по переводу» (https:%%//%%door43.org/en/obs/notes) и «Слов для перевода» (https:%%//%%door43.org/en/obe/kt/home) разработанных специально для «Историй открытой Библии», есть также «Замечания по переводу» (https:%%//%%door43.org/en/bible/notes/home) и «Слова для перевода» (https:%%//%%door43.org/en/obe/home), предназначающиеся для перевода Библии.

«Замечания по переводу» - это культурные, лингвистические и экзегетические замечания, помогающие описать и объяснить библейский контекст, который необходимо знать переводчику, чтобы перевести «Истории открытой Библии», а также информирующие переводчиков о разных способах передачи одного и того же значения.

«Слова для перевода» - это ключевые термины, содержащиеся в «Историях открытой Библии», необходимые для правильного перевода. Для каждого значимого понятия приведена небольшая статья, а также перекрёстные ссылки с теми местами, где использованы данные термины в «Историях открытой Библии» или самой Библии. Они предназначены для того, чтобы показать переводчику иные способы употребления важных слов и чтобы помочь правильно перевести эти слова другом контексте.

После того, как вы ознакомитесь с «Замечаниями по переводу» и «Словами для перевода», вы будете готовы к созданию качественного перевода.

Убедительно просим вас использовать данные материалы («Замечания по переводу» и «Слова для перевода») в ходе вашей работы над текстом!
