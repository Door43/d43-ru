===== Установка текстового редактора =====

This module answers the question: Как установить текстовый редактор?\\
Before you start this module have you learned about:
  * [[ru:ta:vol1:process:platforms|Выбор платформы]]

==== Загрузка исходного текста «Историй открытой Библии» ====

  * Перейдите к Get Open Bible Stories Word Template (Получить текстовый шаблон «Историй открытой Библии»)
  * Выберите ваш исходный язык (тот язык, с которого вы будете осуществлять перевод) и нажмите Save (Сохранить). (Замечание: Если вы выберите Add all images in the file (Добавить все изображения в файл), размер загружаемого файла сильно увеличится). Шаблон будет загружен как файл в формате .docx.
  * Откройте этот шаблон в выбранном вами текстовом редакторе.

==== Загрузка вспомогательных материалов для перевода «Историй открытой Библии» ====
