===== Введение в «Руководство по работе с материалами» =====

This module answers the question: Что представляет собой «Руководство по работе с материалами»?\\
Before you start this module have you learned about:
  * [[ru:ta:vol1:intro:finding_answers|Получение ответов]]

==== Добро пожаловать ====

«Руководство по работе с материалами» - это практический справочник, который информирует переводчиков о том, какие шаги должны быть предприняты от начала до конца и как их осуществлять. Если «Руководство по переводу» представляет собой пособие по теории перевода, «Руководство по редактированию» - пособие по теории редактирования, «Технологическое руководство» затрагивает вопросы о том, как использовать определенные технологические разработки, то «Руководство по работе с материалами» предназначено для того, чтобы показать команде переводчиков пошаговый процесс, которому она должна следовать. В данном пособии описывается весь процесс, начиная с личной регистрации и заканчивая публикацией переведенного и отредактированного содержания.
