==== Как обновлять проекты в tS? ====

Для обновления информации, связанной с проектом, на вашем устройстве выполните нижеперечисленные шаги. Программа translationStudio обновляется каждый раз при открытии приложения.

Замечание: Для данного процесса требуется подключение к Интернету.

Для обновления проектов в translationStudio сделайте следующее: - В разделе Translation Projects (Проекты для перевода) нажмите на три точки слева. - В раскрытом списке нажмите Update Library (Обновить библиотеку). - Для данного процесса требуется подключение к Интернету. Для продолжения нажмите Yes (Да). Нажмите No (Нет) для выхода. - Нажмите на «Историях открытой Библии» или книге Библии. - Выберите язык. Откроется другое окно; выберите или отмените загрузку. - Когда загрузятся все предпочтительные языки, нажмите на левую стрелку в верхней части окна, чтобы вернуться в раздел Translation Projects (Проекты для перевода).
