===== Создать клавиатуру в tK =====

This module answers the question: Как создать клавиатуру в tK?\\
Before you start this module have you learned about:
  * [[ru:ta:vol1:tech:tk_intro|translationKeyboard (клавиатура перевода)]]

==== Начало работы ====

  * Решите, для какого языка вы собираетесь создать клавиатуру
  * Перейдите по ссылке tk.unfoldingword.org/users/sign_in
  * Введите свое имя и пароль

==== Поиск клавиатуры ====

Проведите поиск среди имеющихся клавиатур, чтобы удостовериться, что язык еще не был создан или начат. Доступны три варианта:

  * Клавиатура уже была создана, и это сделали вы!
  * Если клавиатура начала создаваться, но не была завершена, вы можете: Подождать и посмотреть, продолжается ли работа или Лично продолжить работу по созданию клавиатуры и завершить ее.
  * Если желаемая клавиатура не доступна, мы предлагаем вам следующее: Попросите запросчика предоставить вам больше информации о клавиатуре/языке желаемой специфики, например, о символах, нужных для клавиатуры Попробуйте поискать среди уже имеющихся клавиатур с помощью поисковой системы Создайте похожую клавиатуру с небольшими отличиями, с помощью информации, полученной в ходе поиска.

Или

  * Найдите Юникод или сайт преобразования формата Юникода с нужными вам символами.

Далее:

==== Создайте или редактируйте клавиатуру в приложении tK ====

На сайте http:%%//%%tk.unfoldingword.org/ выберите желаемую клавиатуру (если такая имеется) или нажмите на кнопку Create New Keyboard (Создать новую клавиатуру)* в верхней части страницы:

  * Появится окно Create New Keyboard (Создать новую клавиатуру).
  * В поле ввода введите название языка клавиатуры
  * Подберите код языка (ISO), набрав первые три-пять букв нужного вам языка и выберите соответствующий код из списка (для этого необходимо ввести языковой код региона).
  * Тип клавиатуры: Подойдет большинство четырехрядных клавиатур Android, но доступна также и пятирядная вместе с операционной системой для мобильных устройств.
  * Нажмите Go (Перейти) Замечание: Если клавиатура уже была добавлена, выберите вам нужную из списков Languages with Keyboards (Языки с клавиатурами) или Recently Updated Keyboards (Недавно обновленные клавиатуры). Замечание: В конце каждого ряда можно найти знаки «+» и «-». С помощью данных кнопок можно добавить или удалить клавиши с символами в каждом ряду.

==== Редактирование и добавление символов на клавиатуру ====

В следующем окне нажмите кнопку Edit Keyboard (Редактировать клавиатуру) в правом верхнем углу экрана.

Все клавиши будут пустыми до тех пор, пока вы не заполните их нужными вам символами или Юникодом.

==== Введите символы нужного вам языка в tK ====

  * С помощью мыши выбирайте клавишу, которая должна быть отредактирована.
  * Выберите желаемые символы Default (По умолчанию) и Shift (Перемещение) или символы из кодировки UTF-8 (Юникода).
  * Нажмите Save Changes (Сохранить изменения).
  * Если для клавиши требуется дополнительный символ, сделайте следующее: Выберите клавишу и один раз нажмите кнопку Longpress Character (Длительное нажатие на символ) для добавления дополнительного символа. Вы не сможете вводить новые символы в длинные клавиши, пока программа сохраняет клавиши. Вернитесь обратно и введите желаемые символы из UTF-8 (Юникода).
  * Нажмите Save Changes (Сохранить изменения).
  * Замечание: Юникод U+00xx обычно переводится в UTF-8 “xx”. Например: Юникод “U+0d38” аналогичен UTF-8 “d38”. И еще: Если вам удалось найти верную таблицу или символ Юникода для определенного языка, вы сможете скопировать и вставить символы или коды из документа, созданного в программах Word, Excel, Adobe PDF, в клавиши клавиатуры в режиме Edit (Редакировать).
