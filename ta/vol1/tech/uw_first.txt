===== Первое использование unfoldingWord =====

This module answers the question: Как использовать unfoldingWord в первый раз?\\
Before you start this module have you learned about:
  * [[ru:ta:vol1:tech:uw_app|Мобильное приложение unfoldingWord (Раскрывающееся Слово)]]
  * [[ru:ta:vol1:tech:uw_install|Установка unfoldingWord на устройство Android]]

==== Первое использование ====

Когда в первый раз вы откроете приложение unfoldingWord, появится страница для выбора языка. Выберите нужный вам язык. Все истории будут открываться на этом языке. Иные установочные параметры для данного приложения отсутствуют.

==== Уровни редактирования ====

Ниже приводится список языков, дается объяснение трехуровневой системы редактирования. Каждый уровень подробно описывается.

==== Информация о языке ====

Чтобы посмотреть проверочный статус перевода на каждом языке, нажмите на значок i справа от языка. Откроется окно, в котором переводчику будет показан язык перевода, уровень проверки, версия и дата публикации.
