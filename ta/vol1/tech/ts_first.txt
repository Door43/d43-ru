===== Использовать translationStudio в первый раз =====

This module answers the question: Как использовать translationStudio в первый раз?\\
Before you start this module have you learned about:
  * [[ru:ta:vol1:tech:ts_install|Установить translationStudio на устройство Android]]

==== Первое использование tS ====

Выполните следующие действия при первом запуске translationStudio:

  * Для включения планшета нажмите кнопку сбоку.
  * Выберите значок translationStudio.
  * Вам нужно прочитать каждое условие использования и согласиться с ним. Нажмите на каждое условие. Прокручивайте текст, чтобы прочитать каждое условие. Выбирайте «OK» после прочтения каждого условия. Выберите кнопку I Agree (Я согласен).

Перед открытием translationStudio может начаться автоматическое обновление приложения.
