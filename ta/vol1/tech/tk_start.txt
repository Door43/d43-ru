===== Установка приложения translationKeyboard (Клавиатура перевода) на устройство Android =====

This module answers the question: //Как установить приложение translationKeyboard (Клавиатура перевода) на устройство Android?//

  * //[[ru:ta:vol1:tech:tk_install|Установить translationKeyboard (клавиатуру перевода) на устройство Android]]//

==== Как установить клавиатуру для перевода (tK)? ====

Просмотрите трехминутное видео {{:en:ta:tech:setting_up_translationkeyboard-android_v2.mp4|Setting Up the translationKeyboard}} или выполние следующее: Установите translationKeyboard (Клавиатура перевода), чтобы вы могли вводить текст на родном языке. Нажмите на иконку tK, чтобы запустить программу. Откроется окно с инструкциями по установке. Перемещайте информацию на экране с помощью нажатия или направленного вверх движения пальца.

==== Шаг 1: Добавьте translationKeyboard к системе. ====

  * Нажмите на кнопку «Enable Keyboard» («Запустить работу клавиатуры»).
  * Откроется окно со всеми доступными клавиатурами.
  * Найдите в списке клавиатуру для перевода.
  * Найдите включающийся - выключающийся ползунок справа от названия клавиатуры перевода.
  * Нажмите на кнопку. Кнопка станет зеленой.
  * Откроется предупреждающее окно.
  * Нажмите «ОК» для активизации клавиатуры.
  * Нажмите кнопку «Назад» в меню Android, чтобы вернуться в окно установки.

==== Шаг 2: Установка способа ввода ====

  * Нажмите кнопку «set input method» (установка способа ввода).
  * Откроется окно. Нажмите на круг под translationKeyboard.
  * Окно закроется.

==== Шаг 3: Выберите язык ввода ====

  * Нажмите на кнопку «Input Language» (Язык ввода).
  * Откроется список языков ввода.
  * Двигайтесь вверх-вниз, чтобы увидеть все языки в списке.
  * Найдите клавиатуру для целевого языка.
  * Выберите ее, нажав на кнопку справа от языка.
  * Нажмите стрелку «Назад» для возвращения в установочное меню Android.
