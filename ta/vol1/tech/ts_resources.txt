===== Найти материалы в translationStudio =====

This module answers the question: //Как найти материалы translationStudio?//

Before you start this module have you learned about:
  * //[[ru:ta:vol1:tech:ts_create|Создать и редактировать текст]]//
  * //[[ru:ta:vol1:translate:resources_intro|Ознакомление с материалами]]//

==== translationStudio v2.3 (Студия перевода) ====

  * Для редактирования выберите кнопку Resources (Ресурсы) на верхней панели.
  * Нажмите на слово Notes (Замечания) для открытия «Замечаний по переводу».
  * Нажмите Words (Слова) для доступа к «Словам для перевода».

==== translationStudio v3 ====

  * Нажмите на значок для открытия панели.
  * Выберите слово Terms (Термины) для получения информации о «Словах для перевода»).
  * Выберите слово Notes для доступа к «Замечаниям по переводу».
